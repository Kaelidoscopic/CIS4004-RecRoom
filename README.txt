Project Name: Record Room

Description:
A MERN stack music review application that allows users to browse songs and albums, create playlists, and leave reviews.

How to run backend:
1. npm install
2. npm run dev

How to run frontend:
1. cd client
2. npm install
3. npm run dev

Frontend URL:
http://localhost:5173

Backend URL:
http://127.0.0.1:5000

Technologies Used:
- MongoDB
- Express.js
- React (Vite)
- Node.js

Collections:
- users
- songs
- albums
- playlists
- reviews

Notes:
- Requires a MongoDB connection string in .env
- Do not commit .env or node_modules
